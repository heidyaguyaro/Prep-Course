Variable: una variable es un elemento que me permite guardar un valor para recuperarlo cuando quiera. Para crearla se utiliza la palabra clave var seguida de un espacio y el nombre de la variable. Ese nombre es el que voy a utilizar para hacer referencia a ella cuando la necesite. El valor se lo asigno utilizando el signo =. Ej: var nombre = "Heidy";. En este caso "Heidy" es un String.
Strings: son bloques o cadenas de texto que siempre se definen entre comillas. Este texto puede ser letra, número o espacio Ej: "Heidy";.
Funciones: es un bloque de código que realiza alguna acción. Las funciones sirven para cambiar, calcular o hacer algo con las variables, así como ejecutar códigos y re usarlos cuantas veces queramos. Se escribe con la palabra clave function, luego el nombre de la función, apertura y cierre de paréntesis que va a acontener los argumentos que recibe la función y finalmente llaves que van a contener el código que queremos que se ejcute. Ej: function logHola() {console.log ("hola!); } logHola ();. 
a)Las funciones pueden tener a su vez argumentos, que son como variables que declaramos dentro de la función, y que van a tomar un valor cuando invoque la función. Podemos agregar muchos argumentos colocando una coma entre ellos. 
b)Otra cosa que podemos hacer dentro de una función es una declaración. Una declaración determina el nombre y tipo de datos de una variable u otros elementos. Si queremos que nos devuelva algo, utilizaremos return. Ej: function dsumarDosNumeros (a, b) {var producto = a/b; return producto;} 
Declaraciones if: es un tipo de condicional de control de flujo que me permite agregar lógica al código e ir ejecutandolos segun ciertas condiciones. Es decir, if es un condicional que se convierte en verdadero o flaso, y hace una cosa u otra segun las condicones que le indiquemos mediante expresiones. 
Valores booleanos: son los que tienen 2 ocpiones de ejecución, o es uno, o es otro. Ej: true o false.
